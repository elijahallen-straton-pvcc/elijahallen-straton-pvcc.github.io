Networking & The Internet

Category Terms/Definitions
Network Classifications
    LAN - Local Area Network; Often consists of a collection of computers in a single building
    MAN - Metripolitan Area Network; A network of intermediate size, such as covering a local community
    WAN - Wide Area Network; Links machines over larger distances
Network Models
    Bus Network - A model of network in which nodes are directly connected to a bus (a common half duplex link)
    Star Network - a network model in which every host is connected to a central hub.
Protocols
    CSMA/CD Ethernet - Carrier Sense, Multiple Access with Collosion Detection is a protocol which dictates that each message be broadcast to all the machines on a bus.
    CSMA/CA Wifi - Carrier Sense, Multiple Access with Collosion Avoidance is a protocol where wireless networks adopt the policy of trying to avoid collisions rather than dectecting them
Client/Server Model
    P2P: (peer-to-peer) - A server model which involves processes that provide service and receive it from one another
    Distributed System - A system consisting of software units that execute as processes on differnt computers
    Cloud Computing - A system of computing in which massive pools of shared computers on the network can be allocated for use by a clients as the need becomes present
Combining Networks
    Repeater - A device which passes signals back and forth between two original busses without considering the meaning of the signals it is passing
    Bridge - This device also connects two busses, however it doesnt necessarily pass all messafes across a connection, but rather looks at the destination adress for each message, and forwards each message only when it is destined for a computer on the other side
    Switch - A switch is basically a bridge, just with multiple connections which allows it to utilize several busses rather than two
    Router - Special purpose computers used to forward messages
    Gateway - Often, gateways are reffered to as the "point" where one network is linked to an internet as it serves as a passageway between the network and the outside world
Internet Addressing
    Tier 1 - ISPs (Internet Service Providers) that consist of very high-speed, larger capacity, international WANs
    Tier 2 - ISPs that tend to be more regional and less potent regaurding their capabilities, howver these providers help connect to the tier-1 ISPs
    Tier 3 - also known as an "Access ISP" this provider is essentially an independent internet which can also be reffered to as an "intranet", and is operated by a single authority that is in the buisness of supplying internet access to the homes and buisnesses of individuals
IP Address
    ICANN - An orginization that registers domains of internet operated by any differnt authorities
    DNS - The Domain Name System is, like the title suggests, a hierarchical system in which computers, services, and other internet resources are named and distributed
    TLD: (top-level domain) - A suffix following the period at the end of a link which helps to classify each domain (Ex. com, edu, net, etc)
    Country-Code TLDs - These are similar to regular TLDs, but they are used to specify the country in two letters (Ex.)
Internet Protocols
    FTP - File Transfer Protocol is a protocol to download, upload, and transfer files over the internet from one location to another between computer systems
    HTTP - Hyper Text Transfer Protocol is a protocol that is used to load webpages via hypertext links and is able to tranfer information between networked devices
    SMPT - Simple Mail Transfer Protocol is a protocol which acts as one of the standard electronic mail communication protocols and is used to transmit messages by mail servers
World Wide Web
    URL - A Uniform Resource Locater is the unique address that each document is granted on the web so that it can be located and retrieved
    Client-Side Activities - activities that are performed by a client that require additional activity from the browser
    Sever-Side Activities - activities that are performed by a server that require additional activity from the webserver
Internet Four-Layer Software
    Application Layer - A layer of software where messages originate
    Transport Layer - The transport layer is a layer of software that the application layer uses in order to send and recieve messages over the internet
    Network Layer - The network layer takes packets (segments of small sequence numbers produced by the transport layer) and decides the direction each one should be sent along every step of the packets journy over the internet
    Link Layer - This layer is responsible for both recieving and transmitting the packets
Internet is:
    Packet-Switching Network - A type of communications network that sends data in the form of packets
    Fault Tolerance - a property of a system which enables it to continue its operations properly in the event that one or more of its components may become faulty
    Redundancy - A system in which a duplicate of a component is made so that in the case of a failure there will be a backup to continure operations
Cybersecurity
    Malware - Malware is malicious software
    Virus - a virus is a type of malware that infects the computer by putting itself into programs that already exist in the computer
    Worm - a worm is an autonomous form of malware that transfers itself through a network, and may forward itself to other computers
    Trojan Horse - a program which is disguised as a desirable program which when imported by the user the trojan will have access to the computer
    Spyware - a form of malware which collects information about activities the user of the computer may do, and then reports the information back to the instigator of the attack using spyware
    Phishing - technique of obtaining information by asking a user misleading or threataning questions in some instances to obtain user information
    DDos Attack - an attack used on computers in a system in which the computer is flooded with messages
Protection
    Firewall - A program designed to filter traffic and prevent malware on a network system
    Spam Filter - a type of firewall designed to block unwanted emails
    Proxy Server - a software unit which serves as a intermediary between a client and a server with the clients protection in mind
    Antivirus Software - A software used to detect and remove the presence of viruses and infections that are known
    Cryptography - the study of both sending and recieving messages in a secure way while there are present adversaries
    Public Key Encryption - a field of cryptography that involves both the encryption and decryptions, as well as the algoritms are know, but the encoded message is not
