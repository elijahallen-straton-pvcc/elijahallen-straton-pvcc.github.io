Operating Systems Terminology

Term Definition
Operating System The software that controls the overall operation of a computer.
Job Queue A storage organization in which jobs are ordered in first-in, first-out (FIFO).
FIFO An abbreviation which stands for "first-in, first-out."
Interactive Processing A feature that new programs have in which a program being executed is able to carry on a dialogue with the user through remote terminals.
Time-Sharing A feature which allows operating systems to provide service to multiple users at the same time.
Multiprogramming An implementation of time sharing in which time is divided into intervals and the execution of each job is restricted to an interval at a time. At the end of each interval the current job is temporarily set aside for the next in queue to execute during the next interval.
Application Software The programs performing tasks particular to the machines utilization.
System Software Software that performs tasks that are common to computer systems in general, and sets a foundation for the application software to operate within.
Utility Software Programs for performing activities which are fundamental to computer instillations, but are not included in the operating system.
Graphical User Interface (GUI) A user interface in which fies and programs are represented pictorially under icon images.
Shell Older user interfaces which did not utilize images, but rather communicated through textual messages using a keyboard and monitor screen.
Kernel The internal part of an operating system which contains software components that perform the very basic functions required by the computer installation.
Device Driver A software component which communicates with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager A software component whose job is to coordinate the use of the machines mass storage facilities.
Virtual Memory Using a tehnique called paging in which the memory manager rotates data from main memory to mass storage which creates the illusion of more storage by reserving space on a magnetic disk. There, it records the bit patterns that would be stored in main memory, and divids the data into uniformly sized units called pages. By shuffling these pages back and forth between main memory and mass storage, a larger "fictional" memory space is created which is known as virtual memory.
Boot Strapping An operation performed by the computer every time it starts that transfers the operating system from mass storage to main memory.
ROM A place which contains a collection of software routines for performing fundamental input/output activities.
Deadlock The condition where two or more processes are blocked from progressing becasue each is waiting for a resource that is allocated for another.
Spooling A technique where the Operating System holds data for an output at a later but more convienient time.
Super User (or Administrator) Somone who gains highly privileged access to the operating system by identifying themself as the administrator during the login proceedure.
Multi-Factor Authentification A proccess which required more than one form of authentification in order to access an account, such as email or text confirmation.
